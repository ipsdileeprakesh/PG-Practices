					POSTGRES INSTALLATION

1. mkdir /home/postgres -->Create one Directory (postgres)in linux
2. cd /home/postgres
3. copy the postgres version link from websites
4. Use --> wget(webget)+paste postgres version link
5. ls -l --> for the list of files
6. tar -zxvf postgresql-9.6.1.tar.bz
7. cd postgresql-9.6.1
8. mkdir /home/postgres/compiled
9. ./configure --prefix=/home/dileep/postgres/compiled --> here ./ means in this path
